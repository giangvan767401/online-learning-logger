# Online Learning Logger - Thu log sinh viên để train AI dự báo rớt môn

## Chạy local
1. `npm install`
2. `node server.js`
3. Mở http://localhost:3000

## Deploy miễn phí (khuyên dùng Render.com)
1. Tạo tài khoản https://render.com
2. New → Web Service → Connect GitHub repo này
3. Build Command: `npm install`
4. Start Command: `node server.js`
5. Đợi 2 phút → xong!

File log sẽ nằm trong thư mục `/logs` → tự động tạo file mới mỗi ngày:  
`learning_log_2025-12-09.xlsx`

Chúc bạn thu được thật nhiều dữ liệu chất lượng để train model AI dự báo rủi ro rớt môn sớm!
